# 🧮 Mathetrainer von Lias

Ein interaktiver Mathetrainer als Streamlit-App – ideal zum Üben von Addition, Subtraktion und dem Einmaleins.

## Features

- ➕ Addition (Zahlen 1–100)
- ➖ Subtraktion (Zahlen 1–100, immer positives Ergebnis)
- ✖️ Einmaleins (Zahlen 1–10)
- Sofortiges Feedback nach jeder Antwort
- Auswertung mit Prozentwert am Ende
- Jederzeit abbrechbar


## Lokale Installation

```bash
pip install -r requirements.txt
streamlit run mathetrainer.py
```
